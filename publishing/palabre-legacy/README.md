# Palabre Legacy - The Legacy Cycle

This is a GitHub Pages site for Palabre Studios' The Legacy Cycle book series.

## To Deploy

1. Go to repo Settings
2. Enable GitHub Pages
3. Set source to "main branch / publishing"

## Or deploy instantly using:

```bash
# Using surge.sh (no login required)
npx surge . palme.app
```
