## How To Use This File
This file should log every **kill / revise / scale** call with the actual evidence that justified it.

### Required fields for each future entry
- date
- asset / offer name
- decision: **kill**, **revise**, or **scale**
- owner
- channel(s) tested
- traffic / attention signal
- conversion signal
- refund / complaint signal
- AOV / margin notes
- next action

| Date | Decision | Owner | Rationale / Numbers |
|------|----------|-------|---------------------|
| 2026-03-20 | Revise Blade AI Prompts landing page | Alice | CTR healthy, checkout starts weak, proof thin, no bundle path yet |
