## Non-Negotiable Operating Standards
- no product is “finished” unless it passes all five gates:
  - Demand
  - Offer
  - Funnel
  - Economics
  - Scale
- Alice scores each gate **0–5**
- **no scale order if any gate is below 4**
- the old **20 features / 2M tokens** doctrine is revoked
- new doctrine: **one irresistible promise + one trustworthy page + one working checkout + one next-offer path + one measurable acquisition source**
- 100x value must be explicit, not implied
- profit density beats token burn
- no live-link vanity without measurable economics
